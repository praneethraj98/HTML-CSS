png - can have transparent images
jpeg - cannot

border - width, style and color have to be specified either individually or all at once

./ -> current directory

border-top : border only at the top
border-radius : makes the border curved
border-top-left-radius : makes left top curved

background-repeat : repeats the image
backgound-position : shifts the image on x-y scale
background-size: cover : fits the entire selection
