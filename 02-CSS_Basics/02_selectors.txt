id's and classes are used when two tags of same type present and you only want one of them to be different feature

classes are preferred for header tags
ids are preferred for div tags ( div tags are unique)

. is used for classes
# is used for ids

Multiple Selectors used when you want same features for 2 tags
eg: #welcome, #about

Nested Selectors are used when you want a partiular section in a div to be in a different type
eg: #welcome p
no comma

Element Selectors for element tags
