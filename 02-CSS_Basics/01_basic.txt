3 ways to implement CSS:
    1. Inline CSS - style tag used inside body tag
                  - not preferred
    
    2. Internal CSS - style tag used inside head tag
            syntax: a { property : value;}
            "a" - selector. eg: h1,h2,id's, classes
             property - eg: color
             value - eg: yellow

    3. External CSS - seperate file created for CSS
                    - link tag used in the html file to link CSS
                    - preferred option
